# Happy Birthday, Khushi Jain 👑

An elegant, interactive birthday page — gallery, a riddle carousel, a red-carpet
walk, AI "star wishes" (real spoken audio), a curated music mix, a geeky wish-bot,
quotes, and a make-a-wish candle.

## Run
Just open `index.html` in any modern browser (Chrome / Safari / Edge).

## Add your own media (optional)
- **Background music:** save `assets/song.mp3` → tap the ♪ button.
- **Video clip:** save `assets/video.mp4` → shows in "A little clip".
- Per-image songs already stream from YouTube — no files needed.

## Tech
Plain HTML/CSS/JS. No build step. Glass + gold "royal" theme, canvas dust &
confetti, Web Speech API voices, IntersectionObserver reveals.
